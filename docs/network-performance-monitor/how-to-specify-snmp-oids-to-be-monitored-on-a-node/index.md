# How To Specify Snmp Oids To Be Monitored On A Node

Monitored OID is specified for a Node by assigning a specific Poller to the Node. The Poller is predefined in code and can be assigned to the Node via SDK using its name. We have also so called UnDP Poller for custom OIDs where you can select OID from MIBs database.

More information about assigning specific Poller to a node is under [How to assign specific poller to a Node](../how-to-assign-specific-poller-to-a-node/) section.

UnDP Poller for custom OIDs where you can select OID from MIBs database is described under [NPM Universal Device Pollers](../npm-universal-device-pollers/) section.
